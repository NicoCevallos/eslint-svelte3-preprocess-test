# eslint-svelt3-preprocess-test

Project to test feature of [eslint-plugin-svelte3](https://github.com/sveltejs/eslint-plugin-svelte3) allowing preprocessing using [eslint-svelte3-preprocess](https://github.com/NicoCevallos/eslint-svelte3-preprocess).


## How to test

Install the dependencies...

```bash
npm install
```

...then start run linting

```bash
npm run lint
```
